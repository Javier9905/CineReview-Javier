Análisis del producto

Problema, solución y valor académico

01

Problema

El usuario necesita comparar películas de forma rápida sin perder tiempo revisando información desorganizada en diferentes sitios.

02

Solución

El catálogo centraliza reseñas, género, calificación, fecha, detalle y favoritos en una interfaz responsive y fácil de navegar.

03

Resultado

El proyecto evidencia construcción front-end, uso de componentes visuales, JavaScript para interacción y Sass para estilos escalables.

Cumplimiento de la actividad

Implementación técnica en la etapa de profundización

Esta actualización del proyecto original integra Sass de forma estructurada y deja el entregable preparado para su publicación en Bitbucket, cumpliendo con los requerimientos centrales de la actividad.

Proyecto base: CineReview Javier
Entorno: HTML, Bootstrap, JavaScript y Sass
Entrega: repositorio remoto y archivo comprimido
Requisito 01

Variables Sass

Se definieron variables para paleta de color, tipografía, radios, sombras y espaciado, facilitando consistencia visual y mantenimiento del proyecto.

Requisito 02

Anidación de selectores

Los estilos de secciones, paneles, tarjetas y navegación se organizaron con selectores anidados para que el código sea más claro, escalable y profesional.

Requisito 03

Interpolación en Sass

Se implementaron dos casos de interpolación con #{$...} para generar clases reutilizables, cumpliendo de forma explícita lo solicitado por la guía.
