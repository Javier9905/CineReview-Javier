Resumen academico del proyecto

CineReview Javier conserva su catalogo interactivo y ahora organiza la interfaz en componentes, rutas y servicios propios de Angular.

  • Proyecto base: catalogo web de resenas cinematograficas.
  • Objetivo: migrar el front-end a Angular manteniendo la experiencia original.
  • Curso: Desarrollo de Software Web Front-End.
  • Evidencia tecnica: componentes, routing, servicios, Sass y localStorage.
  • Resultado: busqueda, filtros, favoritas, detalle y comentarios por pelicula.

Analisis del producto

Problema, solucion y valor academico

01

Problema

El usuario necesita comparar peliculas de forma rapida sin perder tiempo revisando informacion desorganizada en diferentes sitios.

02

Solucion

El catalogo centraliza resenas, genero, calificacion, fecha, detalle y favoritos en una interfaz responsive y facil de navegar.

03

Resultado

El proyecto evidencia construccion front-end con Angular, componentes reutilizables, routing y Sass para estilos escalables.

Implementacion tecnica en la etapa de profundizacion

Esta migracion integra Sass de forma estructurada y organiza el proyecto en componentes Angular.
