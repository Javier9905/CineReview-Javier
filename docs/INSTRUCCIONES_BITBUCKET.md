# Instrucciones para subir el proyecto a Bitbucket

## 1. Crear el repositorio

1. Ingresar a Bitbucket.
2. Crear un nuevo repositorio.
3. Nombre sugerido: `PIC_CineReview_Javier`.
4. Dejarlo público para que el docente pueda revisarlo.
5. No agregar README desde Bitbucket si ya vas a subir este proyecto completo.

## 2. Comandos Git

Abre la terminal dentro de la carpeta del proyecto y ejecuta:

```bash
git init
git add .
git commit -m "Entrega actividad profundizacion CineReview con Sass"
git branch -M main
git remote add origin URL_DEL_REPOSITORIO_BITBUCKET
git push -u origin main
```

Reemplaza `URL_DEL_REPOSITORIO_BITBUCKET` por la URL real que te entregue Bitbucket.

## 3. Verificación antes de entregar

- El repositorio debe estar público.
- El archivo `index.html` debe abrir correctamente.
- El archivo `scss/styles.scss` debe estar incluido.
- El archivo `css/estilos.css` debe estar incluido.
- El enlace del repositorio debe abrir sin pedir permisos.
