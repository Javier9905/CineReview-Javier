## Guion

Hola, mi nombre es Javier Andrés Barrera Cárdenas, estudiante de Ingeniería de Software. En esta presentación voy a mostrar el proyecto PIC CineReview Javier, desarrollado para la Actividad Académica No. 3 de Profundización del curso Desarrollo de Software Web Front-End.

Este proyecto nace del entregable de contextualización, donde se construyó un catálogo web de reseñas cinematográficas. Para esta nueva etapa, el objetivo fue tomar ese proyecto base y actualizarlo aplicando herramientas propias del desarrollo front-end, especialmente Sass y control de versiones con Git y Bitbucket.

El problema que atiende el proyecto es que muchas veces los usuarios quieren elegir una película, pero la información se encuentra dispersa o poco organizada. Por eso, CineReview Javier propone una interfaz clara donde el usuario puede revisar títulos, géneros, calificaciones, fechas, reseñas breves y una página de detalle de cada película.

La solución desarrollada permite buscar películas por título o reseña, filtrar por género, ordenar por fecha o calificación y guardar favoritas durante la navegación. También se agregó una sección de comentarios locales en la página de detalle, como funcionalidad complementaria para mejorar la interacción del usuario.

A nivel técnico, el sitio utiliza HTML semántico, Bootstrap para apoyar el diseño responsive, JavaScript para la interacción y Sass como preprocesador CSS. En el archivo Sass se implementaron más de cinco variables para controlar colores, tipografía, bordes, sombras y espaciados. También se utilizó anidación de selectores para organizar mejor los estilos y dos casos de interpolación para generar clases reutilizables.

Finalmente, el proyecto queda preparado para subirse a un repositorio remoto en Bitbucket, evidenciando el uso de control de versiones con Git. Con esta actualización se demuestra cómo un proyecto inicial puede evolucionar hacia una versión más organizada, mantenible y cercana a un flujo real de desarrollo web.

Muchas gracias.
